📝 To-Do List Project

Este é um projeto de Lista de Tarefas (To-Do List) desenvolvido para colocar em prática conceitos fundamentais de Front-end. O foco foi criar uma interface limpa, intuitiva e que oferecesse uma experiência de usuário (UX) agradável, incluindo suporte a temas.

🚀 Sobre o Projeto

O objetivo principal deste projeto é permitir que o usuário gerencie suas tarefas diárias de forma simples.
- **Funcionalidades: Adicionar tarefas, marcar como concluídas, excluir itens e persistência de dados local.
- **Destaque: Implementação de **Dark Mode** e **Light Mode** com transições suaves.

🛠 Tecnologias Utilizadas

Para a construção deste projeto, utilizei as seguintes tecnologias:

* **HTML5:** Estruturação semântica do projeto.
* **CSS3:** Estilização personalizada, uso de Variáveis CSS e Flexbox para o layout.
* **JavaScript (ES6+):** Lógica de manipulação do DOM, gerenciamento de arrays e armazenamento local.
* **LocalStorage:** Para garantir que as tarefas não sumam ao atualizar a página.
* **Font Awesome:** Ícones vetoriais para uma interface mais moderna.
* **Google Fonts:** Tipografia selecionada para melhor legibilidade.

🎨 Diferenciais de Design
Como Designer, apliquei conceitos de:
- **Hierarquia Visual:** Para destacar as tarefas pendentes das concluídas.
- **Feedback de Erro:** Mensagens personalizadas no container (em vez de alerts) para uma melhor UX.
- **Responsividade:** Ajustes para garantir que a lista seja funcional em diferentes telas.
